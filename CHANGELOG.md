# Changelog

Все значимые изменения этого проекта документируются в этом файле.

## [2.0.0] — 2026‑04‑16
### Added
- Полная интеграция Django с Telegram‑ботом через защищённый REST API  
- Модель `TelegramUser` с полями: ID, username, имя, язык, геолокация, подписка, активность  
- API‑эндпоинты:
  - `POST /api/v1/register/`
  - `GET /api/v1/user/<telegram_id>/`
- Страница погоды `/weather/` (OpenWeatherMap API)
- Авторассылка погоды в Telegram‑боте
- Inline‑кнопки: выбор города, погода в Пскове, помощь
- Обновлённые READMEru.md и READMEen.md
- Новый `run_all.bat`:
  - одно окно
  - процессы в фоне
  - логи в `logs/django.log` и `logs/bot.log`
- Новый `stop_all.bat`:
  - безопасная остановка Django и бота
  - расчёт времени работы процессов (uptime)
  - UTF‑8 вывод

### Changed
- Улучшена структура проекта (ARCHITECTURE.md)
- Обновлены скриншоты и разделы документации
- Переработана главная README‑страница
- Улучшена админ‑панель (BackupAdminSite)

### Fixed
- Исправлены пути в bat‑скриптах
- Исправлены мелкие ошибки в шаблонах и стилях

### Removed
- Старые версии run_all.bat и stop_all.bat
- Устаревшие фрагменты документации
